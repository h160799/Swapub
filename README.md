# MineForYours
有別於普通的電商拍賣平台，這是一款以交換為設計概念的App，卻不侷限以物易物的交易模式，
顛覆你的想像，找到你心中最棒的寶物!!

### Features 功能介紹
* 首頁
  * 顯示所有上架物件
  * 可依使用者所設定之所在地區，篩選出該地區之物件
  * 點擊放大鏡，透過搜尋引擎輸入關鍵字，快速找到符合的物件
  * 點擊右下按鈕，開啟特殊瀏覽功能，左右滑動菜單模式，快速將物件加入我的最愛
* 物件頁
  * 每一物件之資訊及交易方式，當使用者按下「我有興趣交易」按鈕時，系統自動開啟聊天室
* 聊天室
  * 顯示所有聊天室及最後一則聊天訊息
  * 點擊進入，即可與物件上架者聊天
* 聊天
  * 當使用者與物件上架者達成交易共識時，可點擊「擬定交易」，並輸入雙方之交易細節，等待物件上架者確認交易
  * 物件上架者，接收「確認交易」通知，並同意交易時，雙方交易即完成
* 許願動態
  * 顯示所有「MineForYours」使用者的許願
* 個人頁
  * 個人化管理（我的上架物件，我的收藏、我的社團）
  * 顯示個人許願動態
  * 點擊「新增交易物件」即可立即上架物件或發文許願．
* 設定
  * 編輯個人資訊、設定地區
  * 社團瀏覽
  * 隱私權政策

### Implementation 使用技術
* **Program Language** - Kotlin
* **Design Patterns** - MVVM, Observer, Factory
* **Android Jetpack** - ViewModel, Navigation, LiveData, Data Binding, Permissions
* **User Interface** - Activities and Fragment, RecyclerView, ConstraintLayout, GridLayout,DrawerLayout, Bottom Navigation View, CardView, Splash Screen
* **3rd Parties** - Google Login, Glide, Image cropper
* **Data Management** - Firestore, Firesbase Storage, Firebase, Retrofit, Moshi, Cache
* **Integration test** - Mockito, Espresso

### Requirement 環境需求
* Android Studio 4.0.1+
* Android SDK 28+
